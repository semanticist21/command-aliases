# Placement Guide

- Repo-wide durable AI guidance: root `AGENTS.md`.
- Subtree-specific behavior: nearest subtree `AGENTS.md`.
- User-facing docs: `docs/`.
- Repeatable procedures: `docs/playbooks/`.
- Reusable AI workflow: `.agents/skills/<name>/`.
- Cross-project or machine-level memory: `~/.codex/memo.md`.

Prefer the smallest scope that will still help the next agent.
