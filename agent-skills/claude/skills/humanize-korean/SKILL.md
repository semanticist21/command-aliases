# Humanize Korean

한국어 원문을 사람다운 문장으로 다듬되 의미, 사실, 화자, 높임말, 분량과 형식을 보존한다. 자료·인용·수치·전문용어를 지어내거나 검증 없이 바꾸지 않는다.

## 기본 흐름

1. 독자·매체·말투·변경 범위를 확인한다. 예시가 있으면 그 문체를 우선하고, 없으면 자연스럽고 담백한 한국어를 쓴다.
2. 번역투, 반복 접속사, 과장된 수식, 빈 결론, 기계적인 나열, 불필요한 피동·명사화, 영어식 어순을 고친다.
3. 원문의 구체성·논리·단락 순서·존댓말 수준을 지킨다. 의도된 격식, 법률·기술 용어, 인용문은 유지한다.
4. 소리 내 읽어 어색한 호응, 중복, 리듬, 오해 가능성을 점검한 뒤 최소 수정으로 확정한다.

## 모드

- 기본/fast: 바로 다듬고 변경률이 과도하면 보수적으로 되돌린다.
- `--strict`: 중요한 대외문·긴 글은 문체/사실/의미 보존을 별도로 점검한다.
- `--tone`, `--shorter`, `--formal`, `--casual` 등 명시 옵션을 우선한다.

결과는 다듬은 본문을 먼저 제시하고, 필요할 때만 핵심 변경·남은 확인 사항을 한두 줄로 덧붙인다.
